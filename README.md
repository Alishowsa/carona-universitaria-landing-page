# 🚗 Carona Universitária

Landing page para validação de um aplicativo de carona universitária inteligente, com integração ao Google Forms para coleta de respostas.

## 📌 Sobre o projeto

Este projeto foi desenvolvido como um MVP para validar a ideia de um sistema de mobilidade voltado para universitários.

## 🛠 Tecnologias

- React
- Vite
- JavaScript
- CSS
  

## 🚀 Como rodar

```bash
npm install
npm run dev
